alinhamento Lateral

O alinhamento Lateral pode ser feito facilmente com "margin: auto;"
Observe:

alinhamento Vertical

agora para alinhamento vertical devemos deixar o container com posicionamento "Relative" o que ja vem por padrão, o item dentro deve ter o posicionamento "absolute"

então usamos o top, left para definir o alinhamento (top:50%; left:50%)
Observe que ele realmente esta alinhado, mas é a borda do item e não ele em si

para alinhar da forma que queremos deixando o item totalmente alinhado, usamos o "transform:translate(move direita, move baixo)"

aqui foi usado "trasform:translate(-50%, -50%)" dizendo "retorne metade para esquerda e para cima" assim a div se alinha ao centro

pode ser usado tbm o "Display: flex"

aqui foi usado:
{
display:flex;
justify-content: center; (para alinhar hotizontal)
align-items: center; (para alinhar vertical)
}
importante notar:que esses parametros são colocados no elemento pai para alinhar o elemento filho, no caso o elemento pai é o roxo e o filho o amarelo
