Mesclagem de Células

A Mesclagem de células é simples, é fazer uma célula ocupar o espaço de outra, ou seja ela será expandida para duas células, como no exemplo a baixo:

A B
D E F
I

para termos esse resultado, utilizamos dois comandos direto nas células, são eles o colspan para colunas e o rowspan para linhas, devemos entender quando usar coluna e quando usar para linha. é simples, se quisermos expandir para o lado, ou seja, ocupar a coluna ao lado, usamos colspan, caso a expansão seja para baixo, ocupamos a linha de baixo, assim usamos rowspan, observe os exemplos:

colspan

Observe a celula ocupando 2 colunas

X colspan = "2"
X X X
X X X

rowspan

Observe a celula ocupando 2 linhas

X X X
X X rowspan = "2"
X X
