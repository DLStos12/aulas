Tabelas

Hierarquia de tabelas

existe uma certa "Hierarquia" dentro das tabelas
primeiro temos a tabela, dentro temos as linhas e dentro da linha temos titulos / nome de cada uma e dentro de cada linha tambem temos os dados:
TABLE > TABLE ROW > TABLE HEADER > TABLE DATA
Para criar tabelas, usamos a tag table (a tabela em geral) cor roxa
então criamos as linhas usando tr, significa "table row" (cada linha da tabela) cor amarela
para cada dado dentro da linha, usamos td "table data" (cada conteudo dentro da celula) cor azul

Estilização das tabelas

bordas

para colocar bordas em tabelas, nao pode ser usado somente o parametro border na tag table porque ele deixaria borda somente na parte externa da tabela, para bordas em toda a tabela usamos o parametro border na tag td, assim colocamos borda em cada célula observe:

celula1 celula2
celula1 celula2
celula1 celula2
celula1 celula2

no primeiro exemplo temos a tabela por padrão, ela utiliza "border em table"
no segundo exemplo, corrigimos isso utilizando "border em td"

células

podemos perceer que por padrão as células de uma tabela tem uma margem em cada uma, para corrigirmos isso e deixarmos a tabela mais bonita podemos usar border-collapse com o parametro collapse observe a diferença:

celula1 celula2
celula1 celula2
celula1 celula2
celula1 celula2

no primeiro exemplo temos a tabela por padrão, ela utiliza "border-collapse:separate;"
no segundo exemplo, corrigimos isso utilizando "border-collapse:collapse;"
