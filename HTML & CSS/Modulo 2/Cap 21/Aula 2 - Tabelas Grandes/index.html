Escopos de Tabela

Primeiro devemos entender a diferença de "th" & "td".
o "th" seria um titulo, ele diz sobre o que a coluna fala
o "td" é o dado que a célula comporta
Observe o Exemplo:

isso é um th
isso é um td

observe que por padrão o th ja vem centralizado e em negrito, enquanto o td vem na formatação normal do html, a esquerda e letra normal

o scope não tem nenhum resultado visual, ele serve para indicar para onde o titulo aponta, usamos o scope sempre no th das tabelas, ou seja, sempre no titulo;
o scope tem duas variações:
scope = "col": pega o th e diz que "Este th indica a coluna"
scope = "rol": pega o th e diz que "Este th indica a linha"
observe os exemplos:

scope col 1 scope col 2
resultante scope col 1 resultante scope col 2
scope row resultante scope row

observe q o "Scope" não altera nada visualmente, mas semanticamente ele tem suas indicações, caso haja uma pesquisa na tabela, o scope é o que vai indicar onde estão seus dados

Para esse efeito foi usado em CSS o comando de tr:nth-child(2n)
olhando parte a parte vemos tr > :nth-child > (2n)

tr refere o item que será editado
:nth-child é o comando q seleciona elementos com base na posição
(2) refere a posição a ser escolhida, por exemplo de 2 em 2
neste caso pegamos somente o tr da tabela com o id="zebra" e definimos que a cada 2 linhas teria o background cinza:

ou até :nth-child(even) que seleciona as celulas pares

Fixando o cabeçalho

para tabelas com muitas colunas, pode ser que nos esqueçamos que coluna é aquela, então para resolver esse problema podemos fixar o cabeçalho, para isso, devemos seguir os seguintes parametros:
1-tabela tem que ser position:relative
2-a config do cabeçalho deve ser tr > th{position:stick; top=0;} isso faz com que o cabeçalho cole no topo da tela
para o exemplo usamos o id="fixa" no cabeçalho
