Seletores Personalizados

seletor personalizado é qdo utilizamos "#" para id e "." para classes em CSS.
exemplo:
id="principal" -> #principal
class="destaque" -> .destaque

Atenção : id é unico, ou seja, só pode ter um elemento com esse id na pagina.


se eu coloquei o "id=principal" no primeiro h1, não posso utilizar em outros h1, ele até funciona, mas não é correto.


inclusive a imagem da setinha esta usando uma class e não id, ou seja, poderia ter varias imagens com a mesma class.


podemos usar classes em uma frase apenas ou palavra especifica. usando a tag "span".
