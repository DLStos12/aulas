Pseudo Elementos

Pseudo elementos são utilizados para aplicar estilos em partes específicas de um elemento HTML, sem a necessidade de adicionar classes ou IDs extras. Eles são precedidos por dois pontos (::) e permitem estilizar partes como o primeiro caractere, a primeira linha ou inserir conteúdo antes ou depois de um elemento.

No CSS temos:
# = ID
. = Classes
: = Pseudo classes
:: = Pseudo elementos
> = Children

a{
color: red;
text-decoration: none;
/* tira os efeitos do link, no caso, retirou o sublinhado */
}

a:visited{
color: darkred;
}

/* Ao passar o mouse o link fica preto e sublinhado */
a:hover{
color: black;
text-decoration: underline;
}

/* este codigo é qdo clicar fica da cor escolhida */
a:active{
color: yellow;
}
