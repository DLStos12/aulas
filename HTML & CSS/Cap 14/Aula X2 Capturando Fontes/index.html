Aula X2 - Capturando Fontes

Nesta aula, aprenderemos como capturar fontes personalizadas para utilizar em nossos projetos web, garantindo uma tipografia única e atraente.

Passos para Capturar Fontes

para isso usaremos uma extensão do chrome chamada Fontface Ninja, que nos permite identificar e baixar fontes diretamente de qualquer site.

  1. Instale a extensão Fontface Ninja no seu navegador Chrome.
  2. Acesse um site que utilize a fonte que você deseja capturar.
  3. Clique no ícone da extensão Fontface Ninja na barra de ferramentas do navegador.
  4. Identifique a fonte desejada na lista exibida pela extensão.
  5. Baixe a fonte clicando na opção de download fornecida pela extensão.
  6. Inclua a fonte baixada em seu projeto web utilizando a regra @font-face no CSS.
  7. Ou utilize uma fonte do Google Fonts diretamente no seu projeto web.

Capturando fontes em Imagens

Para capturar fontes presentes em imagens, podemos utilizar ferramentas como o WhatTheFont da MyFonts. Basta fazer o upload da imagem contendo a fonte e a ferramenta irá identificar a fonte ou sugerir fontes semelhantes.

podemos tbm usar sites como o Font Squirrel Matcherator e my fonts que funciona de maneira similar ao WhatTheFont, ajudando a identificar fontes a partir de imagens.
