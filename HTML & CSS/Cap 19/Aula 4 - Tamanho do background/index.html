Tipos de fundo

Fundo Fixo vs Não Fixo:

Fundo Fixo (background-attachment: fixed): A imagem fica fixa na janela e não se move quando a página é scrollada. Cria um efeito parallax elegante.

Fundo Não Fixo (background-attachment: scroll): A imagem se move junto com o conteúdo da página ao fazer scroll. É o comportamento padrão.

background-size: contain

Este faz de tudo pra deixar a Imagem completa

background-size: cover

Este faz de tudo pra deixar a Imagem cobrindo a pagina toda
