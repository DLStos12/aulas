Tipos de Background

O codigo

para inserir imagem em background usamos o "background-image: (url"Url_da_imagem")" e por padrão ela vem: "background-repeat: repeat" "background-size:auto" sendo a repetição e tamanho do backgroud, mas podemos modificar de algumas maneiras
Observe a baixo:

background-repeat:repeat

este vem por padrão, o backgroud se repete por toda a pagina


background-repeat: no repeat

Este não repete a imagem, ela aparece apenas uma vez


background-repeat: repeat-x

Este repete a imagem apenas horizontalmente


background-repeat: repeat-y

Este repete a imagem apenas verticalmente
